    Junior Program

    For kids from 4 - 9 years

    Platform

    Elemento Labs Program is based on Arduino, which is an open source hardware platform developed in Europe.

    Most schools in US, Europe, Singapore and China use Arduino as a teaching platform for their students.

    Elemento Boxes

    Delivered every month (for 3 months) containing all equipments.

    Box 1

    1. Basics of Electronics.
    2. Basics of Arduino.
    3. Software Basics.
    4. LED Fundamental Project.
    5. Musical Notes Project.
    6. Make own Piano Project.
    7. Festival Lights Project.
    8. Build a Robo-car from scratch from the parts shipped in the Box.
    9. Perform various software-controlled navigations on the Robo-car.
    10. Understand usage of fundamentals of C language like while loop, functions, variables.
    11. Perform complex manoeuvers of Robo-car by breaking the problem into smaller ones.

    Box 2

    1. Build distance sensing capability in your Robo-car using an Ultrasonic sensor (In the process, understand the physics behind ultrasonic sensor).

    2. Understand usage of Sound sensor by making an LED light up on sound.

    3. Make your Robo-car move on clap sound using the Sound sensor.

    4. Build a complete Virtual Reality (VR) system from scratch !

    5. Develop your own VR simulation which can be observed in the VR hardware built by the student.

    Box 3

    1. Understand the different types of motors that can be used in Robotics.
    2. Use a Servo motor to build a simple project.
    3. Build a complete HUMANOID that can
      - Walk,
      - Dance,
      - Detect obstacles.
